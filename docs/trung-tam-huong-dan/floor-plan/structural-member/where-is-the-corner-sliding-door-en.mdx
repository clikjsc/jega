Hello, corner sliding doors can be designed using finished (pre-made) or custom aluminum doors and windows.

## Method 1: Using Pre-made Corner Sliding Doors

You can directly search for "corner sliding door" in the public library, or search for the specific model code "**13473708**" to find and use a pre-made corner sliding door.

Below is a screenshot from the new version interface illustrating this:

![Searching for corner sliding door in public library](https://ai-study1.3vjia.com/aihouse-web/ce7438e4-d262-4a18-9d40-182cf51adc08.png)

## Method 2: Assembling a Corner Door Combined with a Window

If you need to create a corner door that is combined with a window, follow these steps:

1.  First, place the corner frame. You can find this by searching for the code "**06846919**".
2.  Then, search for the window/door components using codes "**148891349**" and "**148904972**" for assembly with the corner frame.
