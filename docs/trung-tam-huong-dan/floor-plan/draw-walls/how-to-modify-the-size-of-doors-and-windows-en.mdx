---
title: How to modify the size of doors and windows
description: Learn how to modify the size (length, width, and height) of doors and windows in AiHouse's 2D mode.
icon: /images/icons/floor-plan.svg
---

Hello, please find the required model in Doors and Windows - Door/Window. In the 2D mode, drag and drop it onto the wall with the mouse; then click on the door/window and directly enter the values to modify the length, width and height.
