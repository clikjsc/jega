Hello, to create a circular hole, you can choose either finished products, custom ceilings and walls, free modeling, or panel point editing. The steps are as follows:

Method 1: Go to Cloud Materials -> Doors & Windows -> Windows -> Custom-Shaped Window. Drag it into the 2D mode for use.

![Custom-Shaped Window](https://ai-study1.3vjia.com/aihouse-web/c7bd7bcd-2d8e-4798-8ae0-6efb337d77ba.png)

Method 2: You can create a circular hole in the wall through custom ceilings and walls.

![Circular Hole with Custom Ceilings/Walls](https://ai-study1.3vjia.com/aihouse-web/b6c236aa-5028-4f85-9e80-ac62d0b92906.png)

For more details, you can also refer to specific tutorials like the [Chinese Style Partition Wall Design](https://www.aihouse.com/study/detail/Chinese-Style-Partition-Wall-Design-667).
