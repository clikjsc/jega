# Cách thay thế logo trên báo giá

Bạn có thể dễ dàng thay đổi logo công ty trên báo giá trong phần mềm **JEGA** bằng cách làm theo các bước sau:

## Các bước thực hiện

1. Mở báo giá cần thay đổi logo
2. Nhấp vào nút **"Thay thế logo"**

![Thay thế logo trên báo giá](https://jega.vn/upload/photos/shares/829.1.png)

## Yêu cầu về file logo

Để đảm bảo logo hiển thị đẹp và chuyên nghiệp, file logo của bạn cần đáp ứng các yêu cầu sau:

1. **Định dạng file được hỗ trợ:**
   - JPG/JPEG
   - PNG

2. **Kích thước file:**
   - Tối đa 5MB

3. **Tỷ lệ khuyến nghị:**
   - Tỷ lệ 7:3 (chiều rộng : chiều cao)
   - Ví dụ: 700 x 300 pixels

:::tip
Nên sử dụng file PNG có nền trong suốt để logo hiển thị đẹp nhất trên báo giá
:::

## Một số lưu ý

- Logo mới sẽ được áp dụng cho báo giá hiện tại
- Có thể thay đổi logo nhiều lần nếu cần
- Nên chuẩn bị file logo đúng tỷ lệ trước khi thay thế
