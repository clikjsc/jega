title: How to edit wall shapes?
description: Learn how to edit original walls, shape walls, and create protruding wall designs in AiHouse.

In the new version of Design, you can directly edit the original wall, adding the capability to shape the wall and create protruding wall designs. The specific operation steps are as follows:

In the 2D view, select the wall and click on Shape Edit.

![Select Wall and Shape Edit](https://ai-study1.3vjia.com/aihouse-web/9a90315b-95d9-4856-8935-5d95b59d046e.png)

Switch to the 3D view and draw any shape on the wall.

![Draw Shape in 3D View](https://ai-study1.3vjia.com/aihouse-web/1fe4e5af-a2ab-4ace-8c45-c5e6d8fbfbb3.png)

In the properties panel, select "Protrude" and set the parameters.

![Protrude Properties Panel](https://ai-study1.3vjia.com/aihouse-web/ccb90369-30a9-4205-8712-a09ef4a6f31f.png)

Click "Completed" to return to the design interface.

![Completed Wall Shape Edit](https://ai-study1.3vjia.com/aihouse-web/ecd2bcc3-032f-4b3c-962a-1cc72255db8c.png)
