---
title: How to drag and draw rectangles to draw rooms
description: Learn how to draw rooms by dragging and drawing rectangles in AiHouse using the Floorplan tool.
icon: /images/icons/floor-plan.svg
---

Operation can be performed according to the following steps.

1.  In the Floorplan - Draw floorplan - select the rectangular area;

2.  Drag the rectangular area to the design interface in the 2D mode.
